# 迭代路线图

## V0.1 当前版本

- 运营场景选择
- 艺人/粉丝组织输入
- 内容语气选择
- 多平台内容生成
- 发布日历
- 粉丝任务
- 数据复盘建议
- 风险提醒
- 动态音乐视觉界面

## V0.2 内容素材库

- 选题库
- 标题库
- 评论区问题库
- 多语言标签库
- 封面风格模板

## V0.3 数据复盘录入

- 手动录入曝光、点赞、收藏、评论。
- 自动计算互动率、收藏率、评论率。
- 输出下一条内容建议。

## V0.4 粉丝站协作

- 多成员任务分配。
- 发布日历协作。
- 内容审核状态。
- 素材版权提醒。

## V0.5 多平台连接

- 小红书/抖音/TikTok/X 的发布提醒。
- 热词趋势抓取。
- 评论区情绪分析。
- 高互动内容自动沉淀成模板。

## V1.0 商业化方向

- 粉丝站 SaaS 工具。
- 音乐内容运营工具。
- 艺人宣发团队 Demo。
- 海外粉丝社区运营工具。
