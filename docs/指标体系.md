# 指标体系

## 北极星指标

```text
有效内容运营方案生成数
```

有效不是生成一次，而是完成：

```text
生成方案 -> 复制内容 -> 按日历发布 -> 输入数据 -> 获得复盘建议
```

## 漏斗指标

| 阶段 | 指标 | 说明 |
| --- | --- | --- |
| 创建 | 方案生成完成率 | 成功生成方案 / 点击生成 |
| 使用 | 内容复制率 | 复制内容包 / 生成方案 |
| 执行 | 发布日历使用率 | 使用日历任务 / 生成方案 |
| 参与 | 粉丝任务完成率 | 完成任务粉丝 / 触达粉丝 |
| 传播 | 互动率 | 点赞评论收藏转发 / 曝光 |
| 沉淀 | 复盘使用率 | 输入数据复盘 / 已发布内容 |
| 迭代 | 下一轮生成率 | 复盘后再次生成 / 复盘用户 |

## 内容指标

- 曝光量
- 点赞率
- 收藏率
- 评论率
- 分享率
- 完播率
- 有效评论数
- 新粉关注数
