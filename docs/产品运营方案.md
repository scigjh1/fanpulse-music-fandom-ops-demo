# 产品运营方案

## 运营目标

FanPulse 的目标不是替粉丝“喊口号”，而是把追星热情拆成可执行、可复盘的内容运营动作。

核心目标：

- 提升追星内容生产效率。
- 提升粉丝参与感。
- 帮助粉丝站形成稳定发布节奏。
- 通过数据复盘优化下一轮选题。

## 内容运营链路

```text
热点/回归节点
-> 选题策划
-> 多平台文案
-> 视觉建议
-> 发布日历
-> 粉丝任务
-> 数据复盘
-> 下一轮迭代
```

## 用户分层

### 核心粉

特点：

- 愿意参与任务。
- 了解艺人背景。
- 能生产二创或安利内容。

运营动作：

- 提供任务卡、数据复盘表、评论区互动问题。

### 路人粉

特点：

- 对艺人感兴趣但不了解完整背景。
- 更容易被舞台、歌词、视觉和短视频吸引。

运营动作：

- 提供入坑指南、舞台推荐、代表作清单。

### 海外粉

特点：

- 活跃在 X/Twitter、TikTok、Instagram。
- 需要中英双语表达。

运营动作：

- 提供英文短文案、标签、转发模板。

## 运营活动示例

### 回归期粉丝作战板

- D-7：收集入坑故事。
- D-3：发布安利模板。
- D-Day：发布首日任务卡。
- D+1：复盘首日互动数据。
- D+7：沉淀复盘和下一轮选题。

### 一条内容带路人入坑

内容结构：

```text
3 秒钩子 -> 代表舞台/歌词 -> 入坑理由 -> 评论区问题
```

### 多平台安利模板

- 小红书：情绪、故事、图文收藏。
- 抖音/TikTok：3 秒钩子、节奏剪辑、评论互动。
- X/Twitter：英文短句、话题扩散、实时讨论。

## 数据复盘

核心看：

- 曝光
- 点赞
- 收藏
- 评论
- 转发
- 完播率
- 评论区有效互动

复盘重点：

```text
标题有没有钩子？
封面有没有记忆点？
评论区问题够不够具体？
内容是核心粉能懂，还是路人也能懂？
```
